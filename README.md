# Simple Note Taking SPA

A user can:

- Create Notes
- List Notes
- Update a Note
- Delete a note
- Change theme color of displayed notes

## Instructions

1.  Clone this repo
2.  Run `index.html` on browser

## Built With

```
* Vanilla JavaScript, HTML and CSS
```
